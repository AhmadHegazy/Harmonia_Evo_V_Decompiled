Tell HTC（体验日志）是什么？

Tell HTC（体验日志）是指一种在您的 HTC 手机、平板电脑或设备（下称“您的设备”）后台运行的软件，可帮助 HTC 集团及其关联公司（下称“HTC”）从您的设备中收集特定信息（“使用数据”详情见下文“HTC 想要从您手机上收集什么数据？”）。Tell HTC 并不收集您的个人信息，HTC 也不会使用通过Tell HTC 收集到的信息确认您的身份。

本隐私权声明的适用范围。

本声明仅适用于使用数据的收集和使用。其他 HTC 产品随附的其他版本的 HTC（体验日志），另外适用不同的使用和隐私权政策。

数据“使用”包括数据的使用、储存、传输、转移、处理和披露。本声明不适用于任何其他 HTC 产品、功能、应用程序或服务，也不适用于第三方提供的产品、功能、应用程度或服务。

HTC 想要收集使用数据的原因。

HTC 想要收集使用数据的原因在于，如此 HTC 方能根据您及其他用户的使用方式，开发改良版本的功能、产品和服务，使它们的使用变得更简单、直观。例如，对下载到 HTC 手机、平板电脑和设备的应用程序进行总计之后，HTC 便能判断不同应用程序的受欢迎程度，并将此信息用于后续的 HTC 功能、产品和服务。此外，HTC 还可将使用数据用于统计研究，分析您的设备的运作情况。

HTC 从您的设备上收集使用数据会给您产生费用吗？

您的设备上有关数据传输的默认设置是“仅 Wi-Fi”。在此设置下，只有 Wi-Fi 连接可用时，数据才会被传输。HTC 收集服务将设法开始从您的设备上传输数据，如 Wi-Fi 连接可用，数据将被传输。如果不存在 Wi-Fi 连接，数据不会被传输。

不过，您也可以更改您的设备上的数据传输设置，让您的设备通过 Wi-Fi 连接或可用手机网络（如 GSM 或 UMTS）传输数据。如果您将数据传输设置从“仅 Wi-Fi”更改为“所有数据连接”，您将收到提示信息，说明您可能须为使用可用手机网络传输数据而支付费用。HTC 收集服务将设法开始从您的设备上传输数据，如存在数据连接（Wi-Fi 或手机网络），数据将被传输。根据您与“本国”（您与移动网络提供商订立合约并获得“当地”电话号码的国家/地区）移动网络运营商之间数据计划或合约的条款，虽然传输所收集数据可能会占用一点您享有或购买的数据流量（倘若您没有 Wi-Fi 连接），不过，直接因数据传输导致的费用应视您与移动网络运营商或 Wi-Fi 连接提供商签署的合约而定。传输的数据量视您与移动网络运营商或 Wi-Fi 连接提供商之间数据计划或合约的不同而不同。 HTC 不承担您的移动网络运营商或 Wi-Fi 连接提供商因通过Tell HTC（体验日志）传输数据而收取的费用，因此请通过“仅 Wi-Fi”设置避免移动数据收费。使用第三方 Wi-Fi 连接也有可能会产生费用。

您位于“本国”以外的任何国家/地区时，您的设备不会传输任何数据（数据传输将在检测到数据漫游时自动中止）。只您在“本国”有数据连接（Wi-Fi 连接，或连接到手机网络）时，您的设备才会发送所收集的数据。

HTC 想要从您的设备上收集什么数据？

HTC 想要从您的设备上收集如下数据：

  • 您使用您的设备和您的设备上的应用程序（如 Friend Stream 和 Locations）、快捷方式、类别（如联系人、邮件、信息、音乐、HTC Hub 和 日历）、菜单、功能、程序和服务的详细情况，包括您何时访问这些项目、访问方式、访问次数、您在访问和使用这些项目时采取的行为、访问持续时间、您调整这些项目设置的次数和方式以及您将这些项目的快捷方式从您的设备桌面上删除的时间；
  • 您的设备的序列号和 IMEI 串号（采用单向加密的哈希格式）、网络 MCC 和 MNC（所在国家/地区以及网络提供商的代码）；
  • 与您的设备有关的信息，包括您的设备的语言设置、您所处的网络、网络类型以及您的设备是否处于“漫游”状态（即，您位于“本国”之外的任何国家/地区），调制解调器和内存使用情况；
  • 您的设备的位置信息（市、县）；
  • 与您的设备通话（拨出电话、已接电话）次数和时间有关的信息，与通话时信号强度有关的信息（和其他设备的信息）（不过不包括相关来电人、接听人的姓名或号码），您在您的设备上使用语音拨号的次数（不包括接听人的姓名和号码）；
  • 您收发的短信、彩信数，您的设备储存的信息总数（不过不包括发送人、接收人的姓名或号码，也不包括信息内容）；您一次向多个联系人群发信息的次数以及附加至彩信中的内容类型（如照片、视频）（但不包括该等内容的副本）；
  • 同步到您的设备的电子邮件账户数（但不包括电子邮件地址或账户密码），您是否拥有 Microsoft Exchange ActiveSync（“EAS”）账户，您的电子邮件账户设置（仅服务器和端口设置），您各电子邮件账户中的电子邮件数，您电子邮件账户中电子邮件的平均大小，同步账户所需的时间长度（但不包括发件人、收件人、电子邮件内容或附件）；
  • 与您下载有关的信息，包括您下载了什么，什么时候下载，您是否分享了这些下载内容，您是否卸载、删除了下载内容（如是，何时），您对通过 HTCSense.com 账户下载的内容打几分（如适用）；
  • 有关是否启用了 Wi-Fi 的信息；
  • 您的设备上有几个联系人，您收到了各联系人的多少条信息，与联系人有关的这些信息具体来自何处（即，这些联系人是否是 Facebook 好友或 Gmail 联系人），您在您的设备上设置了几个组（但不包括各联系人的具体资料）；
  • 您的摄像头有无打开，您使用您的设备拍摄的照片和视频数、您的设备储存的照片和视频数、您储存该等照片和视频的文件夹和虚拟文件夹数（但不包括照片或视频副本），您在您的设备上储存的歌曲数；
  • 您的设备的电池状态和 wakelock 信息（wakelock 是安卓电源管理机制，可以阻止您的设备进入低功耗状态）；以及
  • 您使用或访问HTC提供、储存在您的设备上的“帮助”或“显示”文档或视频的次数及持续时间等相关信息（以上信息统称为“使用数据”）。

HTC 收集使用数据的方法

您使用您的设备的方式以及您的行为将被本地记录、储存于您的设备。您的设备将通过安全通道（HTTPS）将所收集的数据从您的设备传输到 HTC 服务器。数据在传输过程中将与单向加密产生的唯一识别码相关联。该唯一识别码不会说明数据来自哪个设备，只是关联来自某一设备的所有数据。

HTC 多久从您的设备上收集使用数据一次？

HTC 大约每 24 小时自动从您的设备上收集数据一次，不过前述频率也可能缩短或延长。从您的设备上收集数据不会让您有所感觉。而且，数据收集的行为也不会对您的设备电池的使用时间产生重大影响。

HTC 会把收集的使用数据储存多久？

HTC 将在收集数据后的一年内汇总数据。在汇总过程中，数据将与首次从您的设备收集信息时所分配的唯一识别码失去关联。HTC 在汇总过程中将删除原始数据。
