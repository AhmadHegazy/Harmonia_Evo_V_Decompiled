Ανήλικοι πελάτες

Η HTC δεν επιθυμεί να συλλέγει οποιαδήποτε δεδομένα από οποιοδήποτε άτομο δεν έχει συμπληρώσει τη νόμιμη ηλικία. Αν δεν έχετε ενηλικιωθεί (δεν έχετε συμπληρώσει τα 18 έτη για τις περισσότερες χώρες) θα πρέπει να πατήστε το κουμπί Απόρριψη στο κάτω μέρος της οθόνης της παρούσας Δήλωσης, υποδεικνύοντας ότι ΔΕΝ συναινείτε στη συλλογή και χρήση των δεδομένων που σας ζητούνται από την HTC στο πλαίσιο της Δήλωσης.

Γιατί συλλέγει η HTC αναφορές σφαλμάτων;

Η HTC πασχίζει να βελτιώσει τις επιδόσεις και την αξιοπιστία του υλικού και του λογισμικού της συλλέγοντας πληροφορίες που μπορούν να χρησιμοποιηθούν για τη βελτίωση των προϊόντων της. Στο πλαίσιο αυτής της προσπάθειας, η Υπηρεσία Αναφορά στην HTC δημιουργεί αναφορές σφαλμάτων που περιέχουν πληροφορίες από τη συσκευή σας, όταν ένα πρόγραμμα, μια εφαρμογή ή ένα τμήμα του υλικού παρουσιάζει σφάλμα. Μπορείτε να επιλέξετε αν θα αποστείλετε ή όχι αυτή την αναφορά σφάλματος στην HTC με τον τρόπο που επεξηγείται παρακάτω.

Τι είναι μια αναφορά σφάλματος;

Όταν παρουσιάζεται ένα σφάλμα, το λογισμικό στη συσκευή σας συλλέγει πληροφορίες για την περιγραφή του σφάλματος, καθώς και άλλες πληροφορίες (για παράδειγμα πληροφορίες για το δίκτυο και την τοποθεσία), οι οποίες μπορούν να βοηθήσουν την HTC να εντοπίσει και να επιλύσει το πρόβλημα. Το περιεχόμενο των αναφορών σφαλμάτων ποικίλει ανάλογα με την αιτία και τη φύση του σφάλματος. Οι αναφορές σφαλμάτων μπορεί να περιλαμβάνουν τους ακόλουθους τύπους πληροφοριών:

Πληροφορίες για το σφάλμα: Πληροφορίες για τον τύπο του σφάλματος και περιγραφή του σφάλματος.

Πληροφορίες για τη συσκευή: Περιλαμβάνουν πληροφορίες για τη συσκευή, όπως το όνομα της συσκευής, το αναγνωριστικό της συσκευής, το αναγνωριστικό κατασκευής, το μοντέλο, το προϊόν και την έκδοση.

Αρχείο καταγραφής σφαλμάτων: Πληροφορίες για την κατάσταση της συσκευής σας κατά τη στιγμή εμφάνισης του σφάλματος, οι οποίες μπορεί να αποδειχθούν χρήσιμες στον εντοπισμό του αιτίου του σφάλματος. Λάβετε υπόψη ότι το αρχείο καταγραφής μπορεί να περιέχει προσωπικά δεδομένα που συλλέγονται ακούσια και δεν πρόκειται να χρησιμοποιηθούν για τον εντοπισμό σας, την επικοινωνία μαζί σας ή για άλλους σκοπούς. Επιπλέον, δεν πρόκειται να μεταβιβαστούν σε τρίτους.

Μπορείτε να προβάλετε τα στοιχεία της αναφοράς πριν την αποστείλετε, εκτός αν έχετε επιλέξει τη λειτουργία αυτόματης αποστολής αναφορών.

Πώς αποστέλλεται μια αναφορά σφάλματος

Την πρώτη φορά που η συσκευή σας ή μια εφαρμογή παρουσιάζει σφάλμα, δεν ανταποκρίνεται ή σταματά να λειτουργεί απροσδόκητα εξαιτίας ενός σφάλματος, θα ερωτηθείτε αν θέλετε να στείλετε μια αναφορά σφάλματος. Μπορείτε να επιλέξετε αν θέλετε να ερωτάστε κάθε φορά που δημιουργείται μια αναφορά ή αν θέλετε να αποστέλλονται αυτόματα οι αναφορές χωρίς προηγούμενη ερώτηση. Αν επιλέξετε την αυτόματη αποστολή αναφορών, οι αναφορές αυτές θα αποστέλλονται στο εξής χωρίς να ειδοποιηθείτε ή να ζητηθεί η συναίνεσή σας. Αν θέλετε να απενεργοποιήσετε τη λειτουργία αυτόματης αναφοράς, μπορείτε να αλλάξετε τη ρύθμιση Αναφορά στην HTC στη συσκευή σας, ώστε να ερωτάστε κάθε φορά πριν την αποστολή μιας αναφοράς.

Την πρώτη φορά που δημιουργείται μια αναφορά σφάλματος, θα σας ζητηθεί επίσης να επιλέξετε τις προτιμώμενες ρυθμίσεις δικτύου, που ελέγχουν τον τρόπο αποστολής της αναφοράς σφάλματος. Μπορείτε να επιλέξετε την αποστολή της αναφοράς όταν υπάρχει σύνδεση δεδομένων ή όταν υπάρχει σύνδεση Wi-Fi ή σύνδεση καλωδίου. Αν δηλώσετε ότι θέλετε να αποστείλετε την αναφορά μέσω της σύνδεσης δεδομένων, ενδέχεται να επιβαρυνθείτε με χρεώσεις. Αν επιλέξετε τη λειτουργία αυτόματης αποστολής αναφορών, το προτιμώμενο δίκτυο που είχατε καθορίσει κατά την επιλογή της αυτόματης αναφοράς θα παραμένει σε ισχύ, εκτός αν το αλλάξετε, προσαρμόζοντας την αντίστοιχη ρύθμιση στην υπηρεσία Αναφορά στην HTC από τη συσκευή σας, ώστε να ερωτάστε πριν την αποστολή μιας αναφοράς. [Μεταβείτε στο Μενού -> Ρυθμίσεις -> Σχετικά με το τηλέφωνο - > Αναφορά στην HTC.]

Αν δεν βρίσκεστε στη χώρα διαμονής σας (δηλαδή το τηλέφωνό σας λειτουργεί με "περιαγωγή") η αναφορά σφάλματος θα αποσταλεί μόνο όταν υπάρχει ενεργή ασύρματη (Wi-Fi) ή ενσύρματη σύνδεση και όχι όταν υπάρχει σύνδεση δεδομένων.

Πώς η HTC χρησιμοποιεί τις αναφορές σφαλμάτων

Η HTC συλλέγει τις αναφορές σφαλμάτων που αποστέλλουν οι πελάτες της, τις αναλύει και χρησιμοποιεί το σύνολο των πληροφοριών, προκειμένου να προσδιορίσει τα αίτια του προβλήματος που αντιμετωπίζουν οι πελάτες της. Οι πληροφορίες των αναφορών σφαλμάτων επιτρέπουν στην HTC και τους προμηθευτές της να εντοπίσουν τα σοβαρότερα σφάλματα και να αναζητήσουν τις αντίστοιχες λύσεις κατά προτεραιότητα. Αν πιστεύετε ότι το πρόβλημά σας οφείλεται στο ασύρματο δίκτυο ή σε μια εφαρμογή ή λειτουργία που παρέχεται από τρίτους, καλό είναι να αποστείλετε μια αναφορά σφάλματος στην HTC, ώστε η HTC να προωθήσει τις πληροφορίες στον ανεξάρτητο πάροχο. Η HTC προωθεί τις πληροφορίες των αναφορών σφαλμάτων σε τρίτους αποκλειστικά για το σκοπό της διόρθωσης των σφαλμάτων και της βελτίωσης των προϊόντων τους. Η HTC θα παρέχει σε τρίτους αποκλειστικά συγκεντρωτικές ή ανώνυμες πληροφορίες που σχετίζονται με σφάλματα και προβλήματα, ενώ οι τρίτοι δεσμεύονται πριν παραλάβουν τα δεδομένα αυτά να τα αντιμετωπίζουν ως εμπιστευτικά. Η HTC δεν μεταβιβάζει τα προσωπικά δεδομένα που μπορεί να έχουν συλλεχθεί από μια αναφορά.

Η HTC δύναται επίσης να αποκαλύψει πληροφορίες που έχουν συλλεχθεί από την Υπηρεσία στο πλαίσιο μιας εταιρικής συναλλαγής, για παράδειγμα στο πλαίσιο μιας συγχώνευσης ή εξαγοράς, κοινοπραξίας, εταιρικής αναδιοργάνωσης, χρηματοδότησης ή πώλησης περιουσιακών στοιχείων μιας εταιρείας (συμπεριλαμβανομένου του νομικού ελέγχου για μια προτεινόμενη εμπορική συναλλαγή). Επιπλέον, η HTC δύναται να αποκαλύψει πληροφορίες σε περίπτωση αφερεγγυότητας, πτώχευσης ή αναγκαστικής διαχείρισης, όπου ενδέχεται να υπάρξει μεταβίβαση πληροφοριών σε τρίτους ως περιουσιακό στοιχείο στο πλαίσιο της συναλλαγής.
